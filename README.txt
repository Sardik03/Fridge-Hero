Fridge Hero - AI Recipe Generator
=================================

Hello! This is my project called "Fridge Hero". It is a simple, dark-themed website 
that helps you generate custom recipes based on 3 ingredients you have in your fridge.
It is powered by Google's Gemini API.

How it works:
1. Open the "index.html" file in any browser (Google Chrome, Safari, Edge, etc.).
2. You will see three boxes for ingredients. Emojis are set by default.
3. Type in your 3 ingredients (for example: eggs, cheese, tomato).
4. The emoji icons will change dynamically as you type! (I even made it match typos, 
 so if you spell "bread" as "beard", it still shows the bread 🍞 emoji).
5. The Gemini API key is already hardcoded in the "script.js" file, so it should work
 immediately. (If you want to use your own key, click the settings gear icon in the 
 top right).
6. Click the orange "Cook Magic" button.
7. A cute cooking pot animation will play, and then your custom recipe will show up 
 with a dish name, description, 3 simple steps, and a plating tip!
8. You can click "Copy Recipe" to copy it, or "Cook Another" to try new ingredients.
